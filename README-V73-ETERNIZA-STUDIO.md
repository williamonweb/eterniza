# Eterniza v7.3.0 — Studio

## Entregue nesta fase
- Novo editor visual em /criar e /studio
- Pré-visualização ao vivo em formato celular
- Temas, fontes, cor de destaque e efeitos
- Editor de nomes, data e carta
- Upload, remoção e limite de fotos conforme o plano
- Seleção de plano real via /api/plans
- Música por link do YouTube
- Salvamento compatível com /api/tributes/draft
- Modais personalizados
- Editor antigo preservado em /criar-classico

## Validação

```
npx prisma generate
npm run build
```
