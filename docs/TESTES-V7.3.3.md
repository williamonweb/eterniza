# Testes v7.3.3

1. Copiar o arquivo `.env` da versão em produção.
2. Executar `npm install`, `npx prisma generate` e `npm run dev`.
3. Abrir `/criar` e selecionar destinatário e plano.
4. Na etapa “Monte a página”, abrir “Assistente IA”.
5. Preencher todos os campos e gerar a homenagem.
6. Confirmar carta, tema e música sugeridos.
7. Testar Melhorar, Continuar, Mais emocionante e Resumir.
8. Confirmar contador de caracteres.
9. Gerar prévia e salvar rascunho.
10. Testar Eterniza Pets e confirmar que continua igual.
11. Executar `npm run build`.
