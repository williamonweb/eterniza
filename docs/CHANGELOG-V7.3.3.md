# Eterniza v7.3.3 — Assistente IA

## Alterações
- Assistente em modal dentro do editor clássico.
- Perguntas sobre destinatário, momento, tom, tamanho e história.
- Geração de carta personalizada.
- Sugestão automática de tema e música.
- Ações rápidas: Melhorar, Continuar, Mais emocionante e Resumir.
- Contador de caracteres da carta.
- Nenhuma alteração no banco de dados.
- Nenhuma alteração no Eterniza Pets.

## Observação
Esta primeira versão usa um motor local seguro para validar o fluxo sem custo de API. A integração com um provedor de IA real pode ser conectada depois sem mudar a experiência aprovada.
