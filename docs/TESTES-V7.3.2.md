# Checklist local — v7.3.2

1. Copie o arquivo `.env` da versão local estável para a raiz desta pasta.
2. Execute `npm install`, `npx prisma generate` e `npm run dev`.
3. Entre em `/criar` e escolha um destinatário.
4. Confirme que um tema adequado já vem selecionado.
5. Troque entre todos os temas e confirme a mudança das cores.
6. Ajuste manualmente as duas cores e confirme que os campos continuam funcionando.
7. Gere a prévia e confirme nome do tema, paleta e aparência.
8. Salve o rascunho, reabra e confirme que o tema permanece.
9. Teste uma homenagem pública já existente para confirmar compatibilidade.
10. Teste `/pets` e o painel Pets: não deve haver nenhuma mudança visual ou funcional.
11. Finalize com `npm run build`.
